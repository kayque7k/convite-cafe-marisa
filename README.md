# Convite de Café ☕

Um convite especial (e ligeiramente irrecusável) para a Marisa Silva tomar um café.

**A graça:** o botão "Não" foge quando ela tenta tocar nele. Depois de 8 tentativas frustradas, ele se rende e vira um segundo "Sim". 😄

- Página única, HTML/CSS/JS puros, zero dependências
- Mobile-first (o botão foge no toque via `touchstart`)
- Confete em canvas ao aceitar + confirmação direto no WhatsApp

**Página no ar:** https://kayque7k.github.io/convite-cafe-marisa/
